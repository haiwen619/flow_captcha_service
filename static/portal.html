文档

接入流程

第一步：注册并登录

创建账号后进入控制台，查看剩余次数和账号状态。

第二步：申请 API Key

在控制台生成自己的 API Key，完整 Key 只显示一次。

第三步：充值与调用

使用 CDK 充值后，即可通过 API Key 正式调用平台能力。

FAQ

常见问题

失败会扣次吗？

不会。当前规则是成功调用才扣减 1 次，失败不扣次。

可以自己申请 API Key 吗？

可以。登录后在控制台的 API Key 页面自助申请。

可以看到平台内部节点信息吗？

不可以。商用用户只看到自己相关的信息，不显示内部运维内容。
